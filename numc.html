Numc

Description

Numc is a slower version of numpy but faster than the naive implementations of matrix operations. Numc can be imported in python and used as a python module with provided methods in Numc. For example, Numc provides matrix addition, multiplication, and power.

What I did

  • Developed a numpy-like Python module with C.
  • Set up the python-C interface by overloading some operators and defining some instance methods.
  • Optimized the operations with Intel SIMD and thread parallelism via OpenMP.
  • Achieved 1900x speed up on pow and 100x speed up on mul compared to naive matrix operations and a top 30 improvement in the leaderboard of the class.
