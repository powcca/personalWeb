PostMe

Visit Site

Description

PostMe is a full stack website that allows users to register their accounts and make user iterations with other users. The web has frontend and backend where frontend is done by react.js and backend is done by node.js. Frontend fetchs data from the API that is provided by the backend, and backend saves data to mongodb.

What I did

  • Developed a website that allows users to register their accounts and make posts to interact with other users.
  • Built a multi-pages website with React.js, Html, CSS, and JavaScript.
  • Built a backend server with Node.js and Express to provide users data via an API and save the data in Mongodb with Mongoose.
  • Deployed the React APP and the server to heroku at https://post-me1.herokuapp.com/
